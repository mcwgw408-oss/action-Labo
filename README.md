# 交流ログLabo

人とのつながりの変化を記録する交流ノートです。

Threads、Substack、生配信、noteなどで増えてきた交流を、誰と話したか、何を感じたか、また見に行きたいかまで含めて残せます。

## 公開URL

https://mcwgw408-oss.github.io/action-Labo/

## 主な機能

- 日付、名前、媒体、交流内容、気づき、感情、メモの記録
- 名前、媒体、メモでの検索
- 登録内容の詳細表示
- 今月の交流件数、媒体別件数、また見に行きたい人数、緊張度の集計
- localStorageによるブラウザ内保存

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

GitHub Pagesへの公開は `.github/workflows/deploy.yml` で自動実行されます。
